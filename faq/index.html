What is the Lynx Framework?

The Lynx Framework is the packaged set of libraries for Lynx components, core Lynx libraries, documentation and Lynx Reference Template(s)

What is a Lynx App Template?

A Lynx Application Template is an ASP.NET Web application that contains all variants to be selected/configured by the associated Lynx Wizard

What is a Lynx App Wizard?

It is a Wizard style desktop application used by a TPM that allows for step-by-step configuration of a Lynx Template to produce a Lynx Application

What is a Lynx Application?

A Lynx Application is the ASP.NET Web App created when running the Lynx Wizard and configuring the combinations of variants supported by the underlying Lynx Application Template

What is a Lynx Application Instance?

This refers to the running instance for a Lynx Application – can support multiple product user experiences via application of Configurable Run-Time Attributes (CRAs) retrieved from the Product Definition System (Jaguar)

What is LynxRoom?

LynxRoom is a Configuration Tool built into Lynx Templates that allows for live/dynamic editing of CSS, Images, I18N string resources and other web resources

How do I install the Lynx Framework?

The Lynx Framework installation instructions are outlined in the Lynx Release Note. This also contains information on how to deploy your Lynx applications on a local IIS server or using Windows Azure Cloud service.
