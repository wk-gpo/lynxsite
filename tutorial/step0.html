Welcome to Lynx Framework tutorial. You will learn how to create web applications based on Lynx. Since best way to learn is to practice, this tutorial will show you step by step how to create simple application based on Velvet interface. You will create application with will let user to search and view documents. In case of any problems during implementation you may download full version of the site and use it as reference in your study.

Lynx Framework is based on AngularJS an HMTL-template-centric solution which promotes close work of web designers and developers. Prior to this tutorial we suggest you to familiarize with that framework using AngularJS Tutorial. We found also video tutorials prepared by John Lindquist on Egghead.io as very useful .

Lynx is a framework build to enable close loop of UX and Developer teams. HTML/CSS designs prepared by UX team constitute basis for Developers. By decoration of HTML with AngularJS and Lynx directives, template provided by UX team becomes working applications. It will allow to use real content thus providing better understanding of application functionality to all parties involved in development cycle: customer, analyst, UX designer, developer.

You will see the overall project lifecycle in following steps of this tutorial. In case of any questions fell free to contact us via email.
