markup is a key

Throughout development cycle application HTML remains the key artefact development work revolves. It allows seemless integration of UX Web designer and application developer work. And what's most important developer responsibility is to add decoration of application HTML prepared by UX specialists. Live Application Editing is provided easily to stremline development process. Just edit your HTML code your preffered IDE and watch results in browser. Developers and designers may use theirs preferred tools.

ux centric

UX team can work directly on the same HTML and CSS sources as development team. UX team can work with their choice of tools, editors, CSS frameworks and libraries.
Developers may immediatly add application logic to prepared templates. Wireframes may be created in HTML enabling quick move from analysis&desing to prototypes and live demos based on real content served by OSA services via Velvet interfaces.

realy rich ui

Lynx dostarcza bogaty zbiór dyrektyw rozszeżających język. HTML, a dzięki temu upraszcza i przyśpiesza proces tworzenia bogatych aplikacji i skomplikowanych layoutów. Dostarczane filtry upraszczają konwersję danych. Elementy logiczne pozwalają na szybkie sterowanie przepływem aplikacji wprost z kodu html. Lynx ma na celu dostarczać ortaz upraszaczać sposób tworzenia skomplikowanych fukcjonalnych komponentów, z których w prosty sposób można złożyć aplikację. Jednocześnie Lynx nie wymusza korzystania z wbudowanych dyrektyw i łatwo integruje się z innymi bibliotekami.
