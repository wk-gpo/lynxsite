lynx is …

... a WK GPO open UI framework, which embraces UX-centric application development technologies like AngularJS and augments them with OSA / WK eco-system integration capabilities and tooling.

based on …

... AngularJS which is a HMTL-template-centric solution promoting close work of web designers and developers. Through bi-directional data binding it makes your code much smaller and readable without need of excesive DOM manipulation or callback functions usage.

giving …

...non-GPO development teams tools and support for local, rapid and flexible development of client-side (Java Sript based) applications on top of OSA 1.2/2.0
Lynx follows on the OSA architecture vision and model of use – it is an OSA-based UX development environment aimed to be used across diverse WK development teams.
