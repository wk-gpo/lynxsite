angularJS - directives

Directives are a way to teach HTML new tricks. You may call it your Domain Specific Language. During DOM compilation directives are matched against the HTML and executed. This allows directives to register behavior or transform the DOM. Directives can use direct DOM creation by Javascript or be templated. Reusable UI Angular's components are created using Directives.
