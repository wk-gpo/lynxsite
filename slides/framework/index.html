lynx framework

  • client side

    Lynx is a client side framework without any server side development requirements. Lynx can be hosted on any web server.

  • modularized

    Lynx is modular whats mean the developers teams can choos only a subset of provided assets. Lynx provides:
    • common module - cross-browser compatibility solutions, misc utilities like a logging etc.
    • services module - the coverage of velvet services
    • core module - which provides specyfic application creation proccess, including global models, routing and misc utils
    • ui module - which provides markup extensions
    • document module - which provides extensions for work with platform (dps) documents

  • end to end proccess

    Lynx provides own workflow for aplication creation proccess with supporting toolset.
