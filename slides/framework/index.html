lynx framework

  • client side

    Lynx is a client side framework without any server side development requirements. Lynx can be hosted on any web server.

  • modularized

    Lynx is modular whats mean the developers teams can choose only a subset of provided assets. Lynx includes:
    • common module - cross-browser compatibility solutions, misc utilities like logging etc.
    • services module - the coverage of velvet services
    • core module - provides specific application creation process including global models, routing and misc utilities
    • ui module - provides markup extensions
    • document module - provides extensions for work with platform (dps) documents

  • end to end process

    Lynx provides own workflow for application creation process with supporting toolset.
