lynx framework

  • client side

    Lynx is a client side framework without any server side development requirements. Lynx can be hosted on any web server.

  • modularized

    Lynx is modular what mean the developers' teams can choose only a subset of provided assets. Lynx functionality is devided into following modules:
    • common - cross-browser compatibility solutions, misc utilities like a logging etc.
    • services - the coverage of velvet services
    • core - specific application creation process, including global models, routing and misc utils
    • ui - Lynx markup extensions - directives, filters & controls
    • document - which provides extensions for work with platform (dps) documents

  • end to end process

    Lynx provides own workflow for application creation process with supporting toolset.
